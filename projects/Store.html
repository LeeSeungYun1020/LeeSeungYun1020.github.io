--- name: Store date: 2020-12-31 summary: 고객, 거래, 상품 DB 관리 웹 어플리케이션 url: https://github.com/LeeSeungYun1020/Store platform: Web, Node.js skills: Node.js, Express, MySQL, HTML, JavaScript, CSS --- 클라이언트는 고객, 거래, 상품 데이터를 보다 쉽게 관리하고 분석하고 싶어한다. 클라이언트가 보다 쉽게 데이터를 관리, 분석할 수 있도록 웹 기반 GUI 시스템을 제공한다. 사용자로부터 고객, 거래, 상품 데이터 파일을 입력 받아 DB에 저장한다. 데이터 속성 기반 검색 기능을 지원한다. 특별한 검색 조건을 지원한다.
